=========
$geometry
=========

.. default-domain:: mongodb

.. query:: $geometry

   .. versionadded:: 2.4

   The :query:`$geometry` operator specifies a :term:`GeoJSON` for a
   geospatial query operators. For details on using
   :query:`$geometry` with an operator, see the operator:

   - :query:`$geoWithin`

   - :query:`$geoIntersects`

   - :query:`$near`
